試合後コメント

堀孝史監督
「前半、特に自分たちの狙ったとおりにゲームを進められていた部分がありましたが、逆に、後半になったときにヴェルディさんのパワーが出てきました。ちょっと自分たちの描いているものとは違うものになりましたが、最後の最後まで選手たちはしっかりと高く意識を持って、やらなければいけないことに取り組んでくれて、それが結果につながったんじゃないかと思っています」

相手チームコメント

川勝良一監督
「前半は、都会のチームと初めてやるような、なんかもう、お嬢さんがやっているようなサッカーで、ハーフタイムに人を変えてポジションを変えて発破をかけて、やっと、らしくなりました。でも決定機が、後半何回もあって、あれだけものにできるチャンスがあって決めきれない。少しでもリーグ戦で昇格できなかったというところで、逆に天皇杯で勝ち進んで、サポーターを含め、いろいろと少しでも返せたらと思って、今日臨んだゲームでしたが、決定機を決めきれなかったです。 後半、しょっぱなとか、途中、何回かありました。途中から選手は良くやりましたが、前半の最初の45分は、まず、もったいないし、単純に負けたということに、非常に悔しいというか、サポーターに申し訳ない、そういう気持ちです」
